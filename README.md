# How to build Compiler

> make

# How to run Compiler
> cd compiler  
> ./bin/alphac /path/to/test

# How to clean Compiler
> cd compiler  
> make clean

# How to build Virtual Machine
> cd vm  
> make

# How to run Virtual Machine
> cd vm  
> ./bin/alpha

By default vm will open the binary.abc which is generated by the compiler
